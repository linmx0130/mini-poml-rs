mini-poml-rs
===
A partial implementation of [POML](https://microsoft.github.io/poml/) purely in Rust.

This project is still in early development. Stay tuned!

## Features 
### Supported features
* Variables
* `if` attribute
* `<let>` for assigning values to variables 
* Render as Markdown
* `<code>` block

### Features in work
* `for` attribute
* Expression evaluation
* `<include>`
